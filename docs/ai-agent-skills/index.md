# 🤖 AI Agent & Skills

探索 AI Agent 和 AI Skills 的最新技术、工具和最佳实践。

## 什么是 AI Agent？

AI Agent 是能够自主规划、决策和执行任务的智能系统。它们不仅能够理解自然语言，还能调用工具、与其他系统交互，完成复杂的 workflows。

## 什么是 AI Skills？

AI Skills 是赋予 AI Agent 特定能力的模块化组件，让 Agent 能够执行更专业、更复杂的任务。

## 本分类涵盖

- **AI Agent 基础**：概念、架构和入门指南
- **Agent 框架**：主流框架对比和使用教程
- **Skills 开发**：如何为 AI Agent 创建和定制 Skills
- **最佳实践**：实际应用场景和案例分享
